Data Source

The dataset used is the PTB-XL ECG Dataset, a large clinical ECG dataset containing diagnostic labels.

Raw Data Characteristics

From the preprocessing pipeline:

Signal shape (after processing): (N, 12, 5000)
DimensionMeaning
NNumber of ECG recordings
1212-lead ECG channels
5000Time samples (10 seconds at 500Hz)

Each ECG sample is: 12-lead waveform × 5000 timesteps

Data Types

Signals → float32
Labels → multi-hot encoded vectors (float32)

Example label vector:

[1, 0, 0, 1, 0]

Corresponding to the diagnostic classes:

['NORM', 'MI', 'STTC', 'CD', 'HYP']

Label Processing

  • SCP diagnostic codes loaded
  • Mapped to diagnostic superclass
  • Encoded using MultiLabelBinarizer()
📌 Task type: Multi-label classification

Signal Preprocessing

1
Bandpass Filtering
butter(4, [0.5, 150])

Purpose — Remove:

  • Baseline drift (<0.5Hz)
  • High-frequency noise (>150Hz)
2
Signal Normalization
(signal - mean) / std

Applied per signal channel. Benefits:

  • Stabilizes gradient updates
  • Improves convergence for deep networks
3
Tensor Shape Conversion
Original: (5000, 12) → Converted: (12, 5000)

Required for Conv1D(in_channels=12)

Dataset Objective

The model uses supervised learning.

Target: ECG → diagnostic label vector

f(x) = y

Where x ∈ ℝ(12×5000) and y ∈ {0,1}5

Loss: BCEWithLogitsLoss — multi-label classification.

Data Augmentation

⚠️ Currently no augmentation is used. This is a limitation because ECG models typically benefit from:

Gaussian noise
Random cropping

Architecture Overview

The encoder is a 1D Residual Network designed for ECG signals.

Model name in code: ECGResNet016

Input(12, 5000)
↓
ResidualBlock12 → 24, stride=2
↓
ResidualBlock24 → 32, stride=2
↓
ResidualBlock32 → 48, stride=2
↓
ResidualBlock48 → 64, stride=2
↓
AdaptiveAvgPool1d(1)
↓
Latent Vector(64D)

Residual Block Design

Each block consists of:

Conv1D(k=7) → BatchNorm → ReLU → Conv1D(k=7) → BatchNorm → + Skip → ReLU

Detailed Architecture

Why Residual Connections?

Residual learning solves:

  • Vanishing gradients
  • Deeper network training instability

Inspired by: Deep Residual Learning for Image Recognition (He et al.)

Latent Space Representation

features = pool(x).squeeze(-1) → Output shape: [batch_size, 64]

Latent representation dimension = 64. This embedding feeds into the classifier head. The encoder acts as: ECG → latent representation.

Activation Functions

Used: ReLU

  • Avoids saturation
  • Efficient gradient flow
  • Computationally cheap

Why This Architecture Fits ECG Data

ECG signals have local waveform patterns and repeating morphology (PQRST). 1D CNNs capture temporal morphology.

Kernel size: 7
QRS complex ≈ 80–120ms
Kernel window ≈ local morphology

Stride-2 downsampling reduces: 5000 → 312. Feature hierarchy emerges.

Evaluation Metrics
